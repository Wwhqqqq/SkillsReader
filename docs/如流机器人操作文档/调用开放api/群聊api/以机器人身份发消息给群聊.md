## **请求体**
**示例：**

```json
{
    "message": {
        "header": {
            "toid": 1671623,
            "totype": "GROUP",
            "msgtype": "TEXT",
            "clientmsgid": 1569398477302,
            "role": "robot"
        },
        "body": [
            {
                "content": "测试",
                "type": "TEXT"
            }
        ]
    }
}
```
**参数说明**：

|参数|类型|参数位置|是否必须|说明|
|-|-|-|-|-|
|toid|long|message.header|是|目标群组的ID|
|totype|String|message.header|是|消息发送的目标类型，此处为 “GROUP”|
|msgtype|String|message.header|是|消息的类型，此处为 “TEXT”|
|clientmsgid|long|message.header|是|客户端生成的一个唯一标识，用于避免消息重复发送|
|clienttime|long|message.header|是|客户端用户排重的唯一标识<br/>如果不传，机器人服务会设置为当前的毫秒时间戳，传0会触发排重机制，建议业务自己维护唯一值|
|role|String|message.header|是|发送者的角色，此处为 “robot”|
|content|String|message.body[0]|是|实际发送的消息内容，此处为测试文本|
|type|String|message.body[0]|是|内容的类型，与上面的 `msgtype` 相对应，此处为 “TEXT”|

## **返回结果**
**示例：**

```json
{
    "code": "ok",
    "data": {
        "errcode": 0,
        "errmsg": "ok",
        "data": {
            "messageid": 1835260055958702611,
            "msgseqid": 300010003,
            "ctime": 1750240379318
        }
    }
}
```
**参数说明**：

|参数|说明|
|-|-|
|errcode|错误代码，`0` 表示成功|
|errmsg|错误信息，`"ok"` 表示操作成功|
|data|包含返回的具体数据项|
|messageid|消息的唯一标识符**，**位于 `data` 对象内|
|msgseqid|消息的序列号，位于 `data` 对象内|
|ctime|消息的创建时间（Unix 时间戳），位于 `data` 对象内|

```json
    "preview":"引用消息原文预览", // 业务方传入被引用消息的预览，如“文字内容……”、[图片]、[文件]
    "imid":"410xxxxxx", // 机器人imid
    "replytype":"1",  //string类型，"1-回复，2-引用"，不可传，默认值"1"
    
}
```

# 消息发送频率限制[​](http://open-demo.dev.weiyun.baidu.com/docs/server/corp_robot-msg-api#%E6%B6%88%E6%81%AF%E5%8F%91%E9%80%81%E9%A2%91%E7%8E%87%E9%99%90%E5%88%B6)
